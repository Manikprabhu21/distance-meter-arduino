# 🚀 Arduino Distance Meter Project

## 📌 Overview
This project measures distance using an ultrasonic sensor and displays the result on an I2C LCD. A buzzer alerts based on object proximity.

## 🛠 Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- I2C LCD Display
- Buzzer
- Jumper Wires

## ⚙️ Features
- Real-time distance measurement
- Displays distance in cm and mm
- Buzzer alert system:
  - Continuous sound (very close)
  - Fast beep (near)
  - Slow beep (moderate distance)
  - No sound (safe zone)

## 🧠 Working Principle
Ultrasonic sensor sends sound waves and measures echo time to calculate distance.
